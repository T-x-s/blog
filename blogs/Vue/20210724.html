    Vue.$nextTick()

    Jokul 7/24/2021 Vue

    # 1、什么是 Vue.$nextTick()？

    定义: 在下次 DOM 更新循环结束之后执行延迟回调。在修改数据之后立即使用这个方法，获取更新后的 DOM。
    简单理解是：当数据更新，DOM 渲染之后，自动执行执行该函数。

    # 2、我们为什么要使用 Vue.$nextTick()?(官网 yyds)

    一个例子让大家明白，如果没有 nextTick 更新机制，那么 num 每次更新值都会触发视图更新，有了 nextTick 机制，只需要更新一次，所以为什么有 nextTick 存在，相信大家心里已经有答案了。

    {
  {
    num
  }
}
for (let i = 0; i < 100000; i++) {
  num = i

    # 3、我们如何使用 Vue.$nextTick()？(网上找的例子)

    更改数据后当你想立即使用 js 操作新的视图的时候需要使用它。

    <template>
  <div class="hello">
    <div>
      <button id="firstBtn" @click="testClick()" ref="aa">{{testMsg}}</button>
    </div>
  </div>
</template>

<script>
export default {
  name: 'HelloWorld',
  data () {
    return {
      testMsg:"原始值",
    }
  },
  methods:{
    testClick:function(){
      let that=this;
      that.testMsg="修改后的值";
      console.log(that.$refs.aa.innerText);   //that.$refs.aa获取指定DOM，输出：原始值
    }
  }
  // 使用Vue.$nextTick()后
    methods:{
    testClick:function(){
      let that=this;
      that.testMsg="修改后的值";
      that.$nextTick(function(){
        console.log(that.$refs.aa.innerText);  //输出：修改后的值
      });
    }
  }
}
</script>
